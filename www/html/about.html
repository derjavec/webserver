About Us

Welcome to our project! This work is part of the Webserv development at École 42 Mulhouse.

Our goal is to create a fully functional web server that showcases modern web technologies, security, and performance.

This project is a collaboration between Denise Erjavec (derjavec) and Jean Ruhlmann (jruhlman), both aiming to understand underlying systems while producing a functional solution.

We believe in the power of code to shape the future, and this project reflects our commitment to learning, growth, and innovation.
